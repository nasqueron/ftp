Nasqueron public FTP server

Welcome to our public FTP server.

If you're lost, maybe are you seeking a file hosted here. You could browse the /pub directory to find it.

If you've uploaded a file, it should be in the /incoming directory.

Server information

AUP

Summary
You can download from /pub.
Respect privacy and use caution browsing /incoming: do it only when you need a document you know it could be there.
You can upload any Nasqueron-related file, but not use it as general file sharing facility.
Download
Files could be provided for general or scoped distribution. If you follow a FTP link from an installation software package or script in /pub directory, this qualifies as an expected and allowed use.
Random browsing in /pub directory is also an acceptable use, as each folder has been identified as suitable for public FTP distribution.
Use caution and respect privacy when you browse the /incoming directory
It could contain private information temporarily stored there to help transfer and communication. Browsing this directory is like wandering in an office: if you genuinely seek a real document, it's probably acceptable to poke around, but checking documents you know it's not related to your current seach would not be acceptable.
Upload
Uploads are restricted to good faith use. You should get the authorization from a Nasqueron member before upload. We enforce this policy socially, no technical enforcement is in place.
This public FTP server isn't a general file sharing facility. Files should be for internal Nasqueron diffusion, or uploaded in the goal to then be moved in /pub for general diffusion related to Nasqueron projects
